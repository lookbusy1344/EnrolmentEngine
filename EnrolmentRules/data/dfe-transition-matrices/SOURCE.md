# DfE transition matrix source

This directory contains a narrow CSV extract from the official Department for Education workbook:

- Publication: Ready reckoner and transition matrices for 16 to 18: 2019
- GOV.UK page: https://www.gov.uk/government/publications/16-to-18-level-3-value-added-ready-reckoner
- Attachment: Transition matrices 16 to 18: 2019
- Original file: TMs_2019A_live_final.xlsx
- Attachment URL: https://assets.publishing.service.gov.uk/media/5e25971940f0b62c45460061/TMs_2019A_live_final.xlsx
- Sheet extracted: All Data - Percentages
- Qualification extracted: 111, GCE A level

The full workbook is intentionally not committed. The CSV keeps only the 14 subjects modelled by this project, one row per subject and
prior-attainment band, with probabilities for A-level grades U, E, D, C, B, A and A*.
