# GCSE grade scale (9–1)

Reference mapping of the reformed numeric GCSE grades to their legacy letter equivalents and pass
status.

| New grade | Old equivalent | Description / pass status |
|-----------|----------------|---------------------------|
| 9 | Above A* | Exceptional performance (top 3% approx) |
| 8 | A* | Very strong A* |
| 7 | A | Strong performance (Grade A equivalent) |
| 6 | High B | Good performance |
| 5 | Low B / High C | Strong Pass (benchmark for many Sixth Forms) |
| 4 | Low C | Standard Pass (minimum level 2 qualification) |
| 3 | D | Below standard pass |
| 2 | E | Well below pass standard |
| 1 | F / G | Minimum achievement |
| U | U | Ungraded / Fail |
